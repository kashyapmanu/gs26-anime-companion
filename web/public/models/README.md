# VRM model placement

A sample VRM file is already included at `sample.vrm` in this folder. The companion
widget loads it by default (`/models/sample.vrm`).

If you want to swap the avatar, replace `sample.vrm` with any free / VRM-license-compatible
VRM 0.x/1.0 model and update `CREDITS.md` with the source, license, and author.

Suggested sources (verify license before use):

- pixiv/three-vrm sample models (VRM License 1.0 where noted)
- VRoid Hub models published under permissive terms (https://hub.vroid.com/)
